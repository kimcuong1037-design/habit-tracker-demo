```
┌─────────────────────────────────┐

│                                 │

│         一  二  三  四  五  六  日│
│  ──────────────────────────────│
│  阅读   ■  ■  ■  ■  ■  □  ◈  │  ← ◈ = 今天（可打卡）
│  运动   ■  □  ■  □  ■  □  ◈  │     ■ = 已打卡
│  喝水   ■  ■  ■  ■  ■  ■  ◈  │     □ = 未打卡

│  早睡   ■  ■  ■  ■  ■  ■  ◈  │
│                                 │

│                                 │
└─────────────────────────────────┘
```

**交互注释：**
- 点击今日列的 `◈` 可直接打卡
- 过去日期的格子不可操作（只读）
- 色块深浅可区分：正常打卡 vs 补卡

| 危险色 | 删除按钮、警告 | 红色 (#EF4444) |
